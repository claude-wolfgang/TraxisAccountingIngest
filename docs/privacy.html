Privacy Policy

Last updated: April 2026

Traxis Accounting Ingest is a private, internal application built and operated by Traxis Manufacturing, Inc. It is not available to the public and is used solely by Traxis staff.

What data we access

This application connects to QuickBooks Online via Intuit's OAuth 2.0 API to:

  • Create Bills (vendor invoices) in the connected QuickBooks company
  • Query the Vendor list for name matching
  • Query the Chart of Accounts for expense categorization
  • Upload PDF attachments to Bill records

How data is used

Data retrieved from QuickBooks Online is used exclusively within the Traxis Accounting Ingest desktop application to match vendors, select expense accounts, and check for duplicate records before creating new entries. No QuickBooks data is shared with third parties, sold, or used for any purpose beyond Traxis's internal accounting workflows.

Data storage

  • OAuth tokens are stored locally in an encrypted environment file on Traxis-owned machines. Tokens are never transmitted to external servers.
  • Extracted document data (vendor names, invoice numbers, amounts) is stored in a local SQLite database on the same machines.
  • Source PDFs are stored in a local Dropbox-synced folder accessible only to authorized Traxis staff.

Data sharing

We do not share, sell, or distribute any QuickBooks data to third parties. The only external services that receive data are:

  • Intuit/QuickBooks Online — to create Bills and upload attachments (the intended purpose)
  • Anthropic Claude API — PDF images are sent for AI-based data extraction (invoice fields only; no QuickBooks data is sent to Anthropic)

Data retention

Local queue records are retained indefinitely for audit purposes. OAuth tokens are refreshed automatically and old tokens are overwritten.

Security

OAuth client credentials are stored in a local environment file excluded from version control. Access is restricted to authorized Traxis personnel. The application runs on Traxis-owned machines behind the company network.
