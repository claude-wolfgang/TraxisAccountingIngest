# Fanuc FOCAS & Machine Monitoring Research

## Overview

Research into Fanuc 0i-F Plus CNC controls for integrating machine monitoring signals with our photo documentation system. Covers digital outputs, FOCAS protocol capabilities, and compatibility with older Fanuc controls (16i being our oldest).

---

## Machine Monitoring Output Options

### 1. Digital Outputs (DO) - Y Addresses

The primary option for triggering external devices like cameras or indicator lights.

| Specification | Value |
|--------------|-------|
| Output voltage | 24V DC (±20%) |
| Max current per point | 200mA (standard) or 2A (2A output module) |
| Saturation voltage | 1V max at 200mA |
| Leakage current when OFF | 20µA or less |
| Delay time | ~50µs driver delay + I/O Link transfer (~2ms) + ladder scan |

### Key Output Signals Available (Fn Addresses)

| Signal | Address | Description |
|--------|---------|-------------|
| OP | Fn000.7 | Automatic operation signal (1 = running) |
| STL | Fn000.5 | Cycle start lamp signal |
| SPL | Fn000.4 | Feed hold lamp signal |
| SA | Fn000.6 | Servo preparation completion |
| RWD | Fn000.0 | Rewind signal |
| RST | Fn001.1 | Reset signal |
| DEN | Fn001.3 | Distribution end signal |
| THRD | Fn002.3 | Threading signal |
| PRTSF | Fn062.7 | Required parts count arrival signal |

### 2. Parts Count & Cycle Complete (Parameters 6700-6713)

- **Parameter 6710**: M code that counts machined parts
- **Parameter 6711**: Number of machined parts counter
- **Parameter 6712**: Total machined parts
- Parts count triggers on **M02, M30**, or a custom M code

### 3. Machine State Monitoring Function (Parameters 11378-11389)

Allows monitoring up to 4 PMC signals:
- Parameters 11378-11381: Set PMC path number for signals 1-4
- Parameters 11382-11385: Set address type (X, Y, G, F, R, etc.)
- Parameters 11386-11389: Set specific address number
- Parameter 24302: Delay time from trigger signal to storing operation history

## FOCAS Protocol

### What is FOCAS?

FOCAS (Fanuc Open CNC API Specifications) is Fanuc's standard protocol for collecting data from CNC machines. It's a set of library files (.dll) that applications can use to retrieve information from the CNC via Ethernet or HSSB (High-Speed Serial Bus).

### Data Available via FOCAS

- CNC state (running, idle, alarm)
- Part count information
- Program name, number, size, and date modified
- Tool and work offsets
- Alarm number and text
- Feed overrides
- Parameters
- Positional data
- Spindle speed
- Modal data
- PMC registers
- Macro variables

### Program Transfer Capabilities

| Function | Description |
|----------|-------------|
| cnc_download | Send NC programs to the CNC |
| cnc_upload | Read NC programs from the CNC |
| cnc_dnc | Send NC commands for DNC operation |

FOCAS can also transfer:
- Tool offsets
- Work offsets
- Parameters
- Macro variables
- PMC data

## FOCAS Compatibility by Control Series

| Control Series | FOCAS Support | Connection Method |
|----------------|---------------|-------------------|
| 30i, 31i, 32i, 35i | ✅ Full FOCAS2 | Ethernet (built-in) |
| 0i-F, 0i-F Plus | ✅ Full FOCAS2 | Ethernet (built-in) |
| 0i-D | ✅ FOCAS2 | Ethernet (usually built-in) |
| 0i-C | ✅ FOCAS1/2 | Ethernet or HSSB |
| 0i-A, 0i-B | ⚠️ Limited | HSSB (requires board) |
| **16i, 18i, 21i** | ✅ FOCAS1/2 | Ethernet or HSSB |
| 15i | ✅ FOCAS1/2 | Ethernet or HSSB |
| 16, 18, 21 (non-i) | ⚠️ FOCAS1 only | HSSB only |
| 15 (non-i) | ⚠️ FOCAS1 only | HSSB only |
| 0M, 0T, 6M, 6T, 10, 11 | ❌ No FOCAS | RS-232C only |

**Key Point**: The "i" series is the dividing line. Our oldest machine (16i) supports FOCAS.

## How to Check if FOCAS is Enabled

### Step 1: Look for Ethernet Settings Screen

1. Put control in **MDI mode**
2. Press **SYSTEM** hard key
3. Press right soft key **(+)** repeatedly until you see **ETHPRM**, **EMBED PORT**, or **ETHER BOARD**
4. If you cycle through all options without seeing ETHPRM, the CNC does NOT have an Ethernet interface

### Step 2: Check for FOCAS2 Settings

1. Press **[EMBED PORT]** or **[ETHER BOARD]** to open Ethernet settings
2. Press **[COMMON]** to display ethernet settings
3. Look for **[FOCAS2]** soft key - if present, you have FOCAS capability

### Step 3: Verify Port Setting

- Default FOCAS port is **8193**
- If TCP Port = 0, FOCAS is disabled

### Step 4: Connection Test

1. Connect machine to network (or directly to laptop)
2. Set machine IP (e.g., 192.168.1.100)
3. Set PC to same subnet (e.g., 192.168.1.50)
4. Ping test: `ping 192.168.1.100`

### Ethernet Port Location

| Control Type | Typical Location |
|--------------|------------------|
| 16i, 18i, 21i | Main electrical cabinet |
| 0i-D, 0i-F | Behind LCD screen |
| 30i, 31i, 32i | Main electrical cabinet |

Port is labeled **CD38A**.

## Implementation Options for Photo Documentation

### Option A: Hardwired DO Signal (Simplest)

1. Machine builder programs ladder rung that turns ON a Y-address output when:
   - Cycle completes (falling edge of OP signal), or
   - M30 executes, or
   - Custom M-code runs (e.g., M50)
2. Wire 24V DO output to relay or directly to device
3. Tablet/camera system detects signal via relay → GPIO adapter

**Pros**: Very reliable, no software complexity  
**Cons**: Requires ladder modification, one wire per machine

### Option B: FOCAS/Ethernet Polling

1. PC or Raspberry Pi polls each machine's status via FOCAS over Ethernet
2. Software detects cycle complete and triggers camera workflow
3. Can also pull work order info if stored in macro variables

**Pros**: No ladder changes, richer data available  
**Cons**: Requires FOCAS license/software, more complex setup

### Option C: Hybrid Approach (Recommended)

Use **hardwired outputs for camera trigger** (simple, reliable, fast) and **FOCAS for data enrichment** (pulling program number, part count, etc.).

Benefits:
- Photo capture doesn't depend on network reliability
- Still get metadata for organizing by work order
- Program number can auto-map to work orders if naming convention exists

## Cost Considerations

- FOCAS activation (if not already enabled): **$500–$2,000** via dealer or FANUC
- FOCAS2 library (A02B-0207-K737): May be available free from local FANUC office
- Fast Ethernet board (if needed for older controls): Additional hardware cost
- Third-party monitoring software: Varies by vendor
