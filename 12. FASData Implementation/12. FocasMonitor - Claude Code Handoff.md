## Known Issue: JSON Deserialization

The original code didn't parse machines.json correctly because of case sensitivity. The fix is in MonitoringService.cs LoadConfig():

```csharp
var options = new JsonSerializerOptions { PropertyNameCaseInsensitive = true };
var config = JsonSerializer.Deserialize<Config>(json, options);
```

Make sure this fix is in place.

## Build for Deployment

Once working locally, build self-contained for the collector PC:

```
dotnet publish FocasMonitor.csproj -c Release -r win-x86 --self-contained true -o publish
copy machines.json publish\
```

The publish folder can then be copied to the collector PC at `C:\FocasMonitor\`
