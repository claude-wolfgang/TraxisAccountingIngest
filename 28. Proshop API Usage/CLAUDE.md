# Project 28: ProShop API Usage

Monitoring and documentation of ProShop GraphQL API usage across the Traxis project ecosystem. Houses recon findings, ProShop vendor correspondence, and future rate-limiting or shared-client work.

## Key Files

- `RECON_PROMPT.md` — Original investigation brief
- `RECON_REPORT.md` — Full findings: per-script API usage, call volume estimates, interval changes, ProShop vendor feedback

## Interfaces

Produces: RECON_REPORT.md (reference documentation for API usage patterns and vendor guidance)
Consumes: All projects that call ProShop GraphQL API (P1, P11, P16, P17, P18, P19, P22, P27)
Contracts: None — this is a documentation/analysis project, not a runtime dependency
